# Automated Task Management System

A Python-based command-line application to automate task 
creation, tracking and management using SQLite database.

## Features
- Add new tasks with priority and due date
- View all tasks
- Update task status
- Delete tasks
- Search tasks by keyword

## Tech Stack
- Python
- SQLite
- Git

## How to Run
1. Clone the repository
2. Run: python main.py
